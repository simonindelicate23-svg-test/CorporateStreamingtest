Set up your music site with zero guesswork

This page walks you through every setup step in plain language. No technical background needed: just follow each card from top to bottom.

Tip: keep this page open while you work through the steps.

1) Install two free tools first

About 10 minutes

Tool A: Node.js — this runs the app on your computer. Install the “LTS” version from nodejs.org.

Tool B: MongoDB database — this stores your songs and album info. If you don't have one yet, create a free MongoDB Atlas account and copy your connection string.

node -v
Run this after install. You should see a version number.

2) Download this project + install it

About 2 minutes

Open Terminal (Mac/Linux) or PowerShell (Windows), then run:

git clone <your-repo-url>
cd Full-generic-music-streaming-app
npm install

3) Run the guided installer

Main setup

This command checks your setup, creates missing files, and tells you exactly what to fix if anything is missing.

npm run fresh-install

4) Fill in your settings file (.env)

Required

Paste your values below and this page builds the exact text you need for your .env file.

MONGODB_URI=
MONGODB_DB_NAME=
APP_BASE_URL=http://localhost:8888
Then paste this into your project's .env file and save.

5) Start the app

Launch
npx netlify dev
Then open http://localhost:8888/player.html.
