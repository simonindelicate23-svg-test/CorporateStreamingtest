Install your site in 4 simple steps

This guide is for the real workflow: your Netlify site uploads media to your hosting via FTP and keeps one canonical JSON record.

Keep this page open. Tick each box as you finish each step.

1) Collect your FTP details

2 minutes

You need these from your hosting provider: FTP host, username, password, and your public media URL.

Example media URL: https://media.yourdomain.com

2) Fill this form once

Auto-build config

Enter your details below. We will generate the exact values needed by Netlify.

APP_BASE_URL=https://your-site.netlify.app
MEDIA_BASE_URL=https://media.yourdomain.com/music
FTP_HOST=
FTP_USER=
FTP_PASSWORD=
FTP_PUBLIC_BASE_URL=https://media.yourdomain.com
FTP_BASE_PATH=uploads
FTP_SECURE=false
ADMIN_PIN=1310

3) Paste into Netlify (single paste)

Required once

Open your Netlify site settings and add the copied values as environment variables.

Netlify → Site configuration → Environment variables

Automatic secret setup is not safe without account-level permission. This one-time paste is the secure approach.

4) Test upload flow

Go live

Open /insert.html, upload one file, and confirm it appears in your hosted media location.

Expected flow: upload in app → file sent by FTP → URL saved to canonical JSON
